# microservice-blog

### Start all k8s service with Skaffold
`skaffold dev`

## K8s command
### Apply file changes
`kubectl apply -f <file_name.yaml>`

### Get pod list
`kubectl get pod`

### Get deployment list
`kubectl get deployment`

### Get service list
`kubectl get service`

### Get logs
`kubectl logs <name>`

### Get describe
`kubectl describe <name>`

### Delete
`kubectl delete <name>`

### Restart
`kubectl rollout restart deployment <deployment_name>`
